# Need, Equity, and Accountability

This is a repository for the paper [“Need, Equity, and Accountability”](https://link.springer.com/article/10.1007/s00355-022-01410-w) by Frauke Meyer, [Jan Romann](https://github.com/JKRhb), Mark Siebel, Stefan Traub, and me.[^1]

[^1]: Bauer, Alexander Max, Frauke Meyer, Jan Romann, Mark Siebel, and Stefan Traub (2022): “Need, Equity, and Accountability. Evidence on Third-Party Distribution Decisions from a Vignette Study”. *Social Choice and Welfare*.
